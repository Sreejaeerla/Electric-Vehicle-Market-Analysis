# Electric-Vehicle-Market-Analysis
European EV market analysis using web scraping, data cleaning, and exploratory analysis to uncover trends in vehicle performance, efficiency, and pricing.

##  Overview

This project analyzes the European Electric Vehicle (EV) market using data collected through web scraping from the EV Database website. The workflow includes data extraction, cleaning, and exploratory data analysis to uncover trends and insights.

##  Objectives

* Scrape EV data from the web.
* Clean and preprocess the dataset using regular expressions.
* Perform exploratory data analysis (EDA).
* Identify trends in range, efficiency, pricing, and vehicle performance.

##  Project Structure


Data_Scraping_EV.ipynb    # Web scraping
Regex_Cleaning.ipynb     # Data cleaning and preprocessing

README.md

##  Key Insights

* EV range and efficiency vary significantly across models.
* Pricing is influenced by performance and battery specifications.
* Different vehicle segments show distinct characteristics.
* Fast-charging capability plays an important role in usability.
